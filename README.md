# PokemonUnite

Projeto criado para fins de estudos do AngularJS. 

## Para executar

- Baixe o repositório para a sua máquina
- Instale o Angular CLI: `npm install --global @angularjs/cli`
- Na pasta do repositório, execute o comando `ng serve`
